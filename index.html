Vaccination wayfinding system
主要目標 / our main goal

本プロジェクトの目的は、東京都墨田区における新型コロナウイルス感染症の予防接種の流れを円滑にするための仕組みを提案することです。

The objective of this project is to propose a way finding system to facilitate people's flow during the coronavirus vaccination process in the Sumida District of Tokyo.

About

墨田区役所

このプロセスには、一定の動きをする多数の人が必要です。特に高齢者は感染症にかかりやすいため、安全な経路を確保し、信号を送ることで安全な距離を保つ必要があります。

This process requires a high amount of persons in constant movement. In particular, so it is necessary to maintain a safe distance through signaling and regulating the safe routes.

collaboration

千葉大学デザイン学科

このプロジェクトは墨田区役所と千葉大学デザイン学科の共同プロジェクトです。提案は空間をナビゲートするための完全な標識システムである。
‍
This project is a collaboration between Sumida Ward office and Chiba University Design Department. The proposal is a complete system of signage for navigating the space.

main requirements

このプロジェクトには主に3つの要件があります。
There are 3 main requirements for this project. 

1. 認識
Recognize

第一の要件は、使用者がワクチン接種プロセス全体を迅速に認識できることである。
‍
The first requierement is that the user should be able to quickly recognize the entire vaccination process.

2. 移動
Move

2つ目の要件は、ユーザーが意識的にある段階から別の段階に移動できることです。
‍
The second requirement is that the users can consciously move from one stage to another.

3. 完了
Complete

そして最後の3つ目の条件は、距離を保ちながらワクチン接種を完了することです。

And finally the third condition is to complete the entire vaccination process while keeping distance.

Let's be safe during vaccination

Design Concept
この設計システムのために,経路と目的地と共に地図上に明確に示された正しい経路を見つける必要があるWayfinding方法論に従った。

For this design system, the Wayfinding methodology was followed, in which it is necessary to find the right path clearly marked on the map with a route and a destination.
