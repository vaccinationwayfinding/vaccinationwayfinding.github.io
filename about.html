Design Concept
設計概念の形態
Morphology of Design Concept

ワクチン接種プロセスの6段階は,プロセスを容易にするために,主要な要素としてステージの名前を使用する代わりに,各段階は色と数に関連する:色に分けられた。
‍
The six stages of the vaccination process were divided into colors: Each stage is associated with a color and a number instead of using the name of the stage as the main element in order to facilitate the process.

カラー選択
Color Selection

色の選択は、6色が選択され、補色が互いに重ならないように構成されたMunsellカラーモデルに基づいています。
‍
Color selection is based on the Munsell color model where six colors were selected and organized so the complementary colors do not overlap with each other.

波形パターンの概念
Wave Pattern Concept

視覚系を標準化するために,波形パターンをすべての図形要素に適用した。
この模様は、段ボールの側面に見える波と、粘着テープを切る際に端にできる波に基づいています。
‍
To standardize the visual system, a wave pattern was applied to all graphic elements. This pattern is based on the waves that can be seen on the sides of the corrugated cardboard and also on the waves that are created on the edges when cutting adhesive tape.

main requirements

3 Basic Elements
There are 3 main requirements for this project. 

1. 用紙フォルダ
Paper Folder

このフォルダーの目的は、6段階のワクチン接種プロセスを説明することです。
‍
The purpose of this paper folder is to illustrate the six-step vaccination process.

2. 誘導線
Induction Line

2つ目の要件は、ユーザーが意識的にある段階から別の段階に移動できることです。
‍
The second requirement is that the users can consciously move from one stage to another.

3. 看板パネル
Sign Panels

ワクチン接種プロセスの各段階で設置される標識パネル。

Sign panels installed at each stage of the vaccination process

element 1

用紙フォルダ
Paper Folder

このペーパーフォルダーのもう1つの重要な機能は、プロセス中に必要な他の文書を保存することであり、ワクチン接種は2つの段階に分けられるため、このフォルダーは次の予約のための文書を保存するためにも使用できます。

Another important function of this paper folder is to store other documents needed during the process and since the vaccination is divided into two phases, this folder can also be used to store the documentation for the next appointment.

Element 2

誘導線
Induction Line

ワクチン接種プロセスの各段階をつなぐ誘導ライン。このラインは50 mm幅のテープで構成され、床にテープで固定する必要があります。
‍
Induction line to connect each stage of the vaccination process. This line consists of a 50mm wide tape that must be taped to the floor.

element 1

看板パネル
Sign Panels

これらの標識は2つのタイプに分けられます。1つ目のタイプは、壁または机に取り付ける必要がある段ボールパネルです。第2のタイプは、床、特に椅子の下に設置しなければならない接着パネルで、第3段階ではワクチン接種を待たなければならない。

These signs are divided into two types: The first type is a corrugated cardboard panel that must be mounted on a wall or desk. The second type is an adhesive panel that must be placed on the floor, especially under the chairs during the third stage of the process where users must wait to be vaccinated.
